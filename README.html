Done'N'Dusted

Your personal task manager to stay productive and organized!

🌟 Features

  • User Authentication (Sign Up, Log In, Log Out)
  • Task Management (Add, Delete, Mark as Completed)
  • Category Management (Add, Delete)
  • Task Due Date Tracking
  • Interactive Calendar for Task Visualization
  • Responsive Design with Bootstrap
  • Dynamic Gradient Animations

🚀 Getting Started

  1. Clone the Repository:
    git clone https://github.com/yourusername/donendusted.git
    cd donendusted
  2. Create a Virtual Environment:
    python -m venv venv
    source venv/bin/activate
    # For Windows use `venv\Scripts\activate`
  3. Install Dependencies:
    pip install -r requirements.txt
  4. Set Up the Database:
    python setup_db.py
  5. Run the Application:
    python app.py
  6. Access the App:
    http://127.0.0.1:5000

🖱️ How to Use

🔐 Sign Up

Navigate to the Sign Up page, create a unique username and password, then click "Sign Up."

🔑 Log In

Use your credentials to log in to your personalized dashboard.

🗂️ Dashboard

Add tasks, set due dates, and organize them by categories. Mark tasks as completed or delete them as needed.

📅 Calendar

Visualize your tasks and deadlines with color-coded indicators.
